# Authentication 

This request  uses   `<Bearer_Token>` from the register API collection.

---

### POST `{BASE_URL}/auth/register`


---

### Request Body

| Field           | Type   | Required | Description                     |
| --------------- | ------ | -------- | ------------------------------- |
| name            | string | Yes      |  Flacko's tech meet up             |
| status          | string | Yes      |  Active                |
| description     | string | Yes      |  Meet up with the tech bros        |
| posterUrl       | string | Yes      |  [image]             |
| price           | string | Yes      |  20  |
| discount        | string | Yes      |  5        |
| type            | string | Yes      | Either "PERSONAL" or "BUSINESS" |


---
