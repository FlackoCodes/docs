### POST /{BASE_URl}/v1/shops/products/product_id

## Authorization
This endpoint requires a Bearer Token for authorization. Include it in the Authorization header

Authorization: Bearer `<your_token>`



### Request Body

| Field           | Type   | Required | Description  / e.g              |
| --------------- | ------ | -------- | ------------------------------- |
| name            | string | Yes      | User's name                     |
| description     | string | Yes      | product description             |
| categories      | Array  | Yes      | Product categories (e.g., ["Fashion"])|
| bannerUrl       | string | Yes      | Main product image URL          |
|gallery          | Array  | Yes      | [https://www.istockphoto.com/stock-photos] |
| stockQuantity   | number | Yes      | Number of  items in stcok       |
| price           | number  | Yes     | Product price                   |
| discount        | string | No       | Discount Value                  |
| currency        | string | Yes      | Currency Code e.g (CAD)         |
| slug            | string | Yes      | casio-f91                       |
| features        | string | Yes      |  Array of feature IDs           |

### Status Codes
- `200 OK` - Product updated successfully
- `400 Bad Request` - Invalid input
- `401 Unauthorized` - Missing or invalid token
- `404 Not Found` - Product not found
