---
title: "List Business"
openapi: "GET /{BASE_URL}/v1/users/{parameters}"
description: "This endpoint lists all businesses or vendors created"
---

Lists all businesses registered on the Cleva platform.  
This endpoint accepts the required parameters for authorization and returns a list of all vendors upon a successful request (`200 OK`).  
A valid Bearer Token must be included in the `Authorization` header to access this endpoint.

### Request Params

| Field    | parameter |
| -------- | --------- |
| isActive | true      |
| page     | 1         |
| size     | 15        |
| filters  | BUSINESS  |

---

---
